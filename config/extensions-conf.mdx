## Overview

The `extensions.conf` file contains the Asterisk dialplan, which defines how calls are processed and routed. It's the core logic of your PBX.

## File Location

```
/etc/asterisk/extensions.conf
```

## Configuration Structure

### Sections

- `[general]` - General dialplan settings
- `[globals]` - Global variables
- `[contexts]` - Dialplan contexts containing extensions

## [general] Section

<ParamField path="static" type="boolean" default="no">
  Whether the dialplan is static (won't change at runtime)
</ParamField>

<ParamField path="writeprotect" type="boolean" default="no">
  Prevent dialplan from being modified at runtime
</ParamField>

<ParamField path="clearglobalvars" type="boolean" default="no">
  Clear global variables on dialplan reload
</ParamField>

## [globals] Section

Define global variables accessible throughout the dialplan:

```ini
[globals]
CONSOLE=Console/dsp
TRUNK=DAHDI/G2
MAIN_OFFICE=4000
```

Access globals with `${GLOBAL(varname)}` or just `${varname}`.

## Dialplan Basics

### Context Syntax

Contexts group related extensions:

```ini
[context-name]
exten => extension,priority,Application(args)
```

### Extension Syntax

```ini
exten => number,priority,Application(args)
```

**Components:**
- `number` - Extension number or pattern
- `priority` - Execution order (1, 2, 3... or use `n` for next)
- `Application()` - Dialplan application to execute

### Priority Shortcuts

```ini
exten => 4000,1,Answer()
exten => 4000,n,Playback(welcome)    ; n = next (auto-increments)
exten => 4000,n,Hangup()

; Same as above with 'same' keyword:
exten => 4000,1,Answer()
same => n,Playback(welcome)
same => n,Hangup()
```

## Pattern Matching

Pattern matching allows extensions to match ranges of numbers:

### Pattern Characters

| Pattern | Matches | Example |
|---------|---------|----------|
| `X` | Any digit 0-9 | `_X` matches 0-9 |
| `Z` | Any digit 1-9 | `_Z` matches 1-9 |
| `N` | Any digit 2-9 | `_N` matches 2-9 |
| `.` | One or more of previous | `_X.` matches 1, 12, 123... |
| `!` | Zero or more of anything | `_9011!` matches 9011, 90111... |
| `[]` | Character range | `_[12-4]XXX` matches 1000-1999, 2000-2999, 3000-3999, 4000-4999 |

### Pattern Examples

```ini
; Match any 4-digit extension
exten => _XXXX,1,Dial(SIP/${EXTEN})

; Match extensions 4000-4099
exten => _40XX,1,Dial(SIP/${EXTEN})

; Match numbers starting with 9 (external calls)
exten => _9.,1,Dial(${TRUNK}/${EXTEN:1})

; Match international calls (9011 + anything)
exten => _9011!,1,Dial(${TRUNK}/${EXTEN:4})

; Match US toll-free numbers
exten => _91800NXXXXXX,1,Dial(${TRUNK}/${EXTEN:1})
```

## Common Applications

### Dial()

Dial one or more endpoints:

```ini
; Basic dial
exten => 4000,1,Dial(SIP/javier)

; Dial with timeout (15 seconds)
exten => 4000,1,Dial(SIP/javier,15)

; Dial multiple endpoints simultaneously
exten => 4040,1,Dial(SIP/javier&SIP/belen,60)

; Dial with options (p = screening)
exten => 4000,1,Dial(SIP/javier,20,p)
```

### VoiceMail()

Send caller to voicemail:

```ini
; Leave message in mailbox 4000
exten => 4000,1,VoiceMail(4000)

; Unavailable greeting
exten => 4000,1,VoiceMail(4000,u)

; Busy greeting
exten => 4000,1,VoiceMail(4000,b)

; With context
exten => 4000,1,VoiceMail(4000@default)
```

### VoiceMailMain()

Access voicemail box:

```ini
; Access own mailbox (authenticated)
exten => *97,1,VoiceMailMain()

; Access specific mailbox
exten => *4000,1,VoiceMailMain(4000)
```

### Playback()

Play a sound file:

```ini
exten => 100,1,Answer()
exten => 100,n,Playback(welcome)
exten => 100,n,Hangup()

; Skip if channel not answered
exten => 1234,1,Playback(transfer,skip)
```

### Answer()

Answer the incoming call:

```ini
exten => s,1,Answer()
```

### Hangup()

Hang up the call:

```ini
exten => s,1,Hangup()
```

### NoOp()

No operation (used for logging/debugging):

```ini
exten => 4000,1,NoOp(Calling Javier)
same => n,Dial(SIP/javier)
```

### Wait()

Wait for specified seconds:

```ini
exten => s,1,Wait(1)           ; Wait 1 second
exten => s,n,Wait(0.25)        ; Wait 250ms
```

### Goto()

Jump to another location:

```ini
; Goto context,extension,priority
exten => 100,1,Goto(reception,s,1)

; Goto extension,priority (same context)
exten => 100,1,Goto(s,1)

; Goto priority (same extension)
exten => 100,1,Goto(5)
```

### GotoIf()

Conditional branching:

```ini
exten => s,1,GotoIf($["${EXTEN}" = "4000"]?yes:no)
exten => s,n(yes),Playback(extension-4000)
exten => s,n,Hangup()
exten => s,n(no),Playback(other-extension)
```

### Background()

Play audio while accepting digit input:

```ini
exten => s,1,Answer()
exten => s,n,Background(main-menu)
exten => s,n,WaitExten()
```

### WaitExten()

Wait for caller to dial an extension:

```ini
exten => s,1,Background(menu)
exten => s,n,WaitExten(10)    ; Wait 10 seconds
```

### Pickup()

Pick up a ringing call:

```ini
; Pick up call to extension 4000
exten => _*8.,1,Pickup(${EXTEN:2})

; Pick up from different context
exten => _*8.,1,Pickup(${EXTEN:2}@empleado)
```

### MessageSend()

Send SIP MESSAGE:

```ini
exten => 4000,1,MessageSend(sip:javier,Centro de Mensajes)
```

## Variables

### Channel Variables

| Variable | Description |
|----------|-------------|
| `${EXTEN}` | Dialed extension |
| `${CONTEXT}` | Current context |
| `${PRIORITY}` | Current priority |
| `${CHANNEL}` | Channel name |
| `${CALLERID(num)}` | Caller ID number |
| `${CALLERID(name)}` | Caller ID name |
| `${DIALSTATUS}` | Result of Dial(): ANSWER, BUSY, NOANSWER, CANCEL, CONGESTION, CHANUNAVAIL |
| `${EPOCH}` | Unix timestamp |

### String Operations

```ini
; Substring: ${EXTEN:start:length}
exten => _9.,1,Dial(${TRUNK}/${EXTEN:1})  ; Strip first digit

; Cut: ${CUT(variable,delimiter,field)}
same => n,Set(user=${CUT(MESSAGE(from),@,1)})

; Filter: ${FILTER(allowed,string)}
same => n,Set(clean=${FILTER(0-9,${INPUT})})
```

### Setting Variables

```ini
exten => s,1,Set(myvar=value)
exten => s,n,Set(TIMEOUT(digit)=5)
exten => s,n,Set(CHANNEL(language)=es)
```

## Special Extensions

### Special Extension Names

```ini
; Start extension (for incoming calls)
exten => s,1,Answer()

; Invalid extension
exten => i,1,Playback(invalid)

; Timeout extension
exten => t,1,Hangup()

; Hangup extension  
exten => h,1,NoOp(Call ended)

; Any digit (when using * in pattern)
exten => a,1,VoiceMailMain()
```

## Context Inclusion

Include one context in another:

```ini
[local]
exten => _XXXX,1,Dial(SIP/${EXTEN})

[longdistance]
include => local              ; Include all of [local]
exten => _9NXXNXXXXXX,1,Dial(${TRUNK}/${EXTEN:1})

[international]
include => longdistance       ; Includes longdistance and local
exten => _9011.,1,Dial(${TRUNK}/${EXTEN:4})
```

## Example Configurations

### Basic Office Extensions

```ini
[general]
static=yes
writeprotect=no

[globals]
CONSOLE=Console/dsp

[empleado]
exten => 4000,1,Dial(SIP/javier)
exten => 4001,1,Dial(SIP/belen)
```

### With Voicemail Fallback

```ini
[empleado]
exten => 4000,1,NoOp(Calling Javier)
same => n,Dial(SIP/javier,15)
same => n,VoiceMail(${EXTEN})
same => n,Hangup()

exten => 4001,1,NoOp(Calling Belen)
same => n,Dial(SIP/belen,15)
same => n,VoiceMail(${EXTEN})
same => n,Hangup()
```

### Voicemail Access

```ini
[empleado]
; Access own voicemail
exten => *4000,1,VoiceMailMain(4000)
same => n,Hangup()

exten => *4001,1,VoiceMailMain(4001)
same => n,Hangup()
```

### Call Pickup

```ini
[empleado]
; Pickup extension - dial *8 + extension number
exten => _*8.,1,NoOp(Picking up call to ${EXTEN:2})
same => n,Pickup(${EXTEN:2})
same => n,Hangup()

; Pickup from different context
exten => _*9.,1,NoOp(Picking up call from other context)
same => n,Pickup(${EXTEN:2}@direccion)
same => n,Hangup()
```

### Ring Multiple Extensions

```ini
[entrantes]
; Ring both extensions simultaneously
exten => 4040,1,NoOp(Calling group 'empleados')
same => n,Dial(SIP/javier&SIP/belen,60)
same => n,Hangup()
```

### Conditional Routing

```ini
[empleado]
exten => 100,1,NoOp(Reception)
same => n,GotoIf($["${EXTEN}" = "4000"]?javier:belen)
same => n(javier),Dial(SIP/javier)
same => n,Hangup()
same => n(belen),Dial(SIP/belen)
same => n,Hangup()
```

### Message Handling

```ini
[mensajes]
exten => _X.,1,NoOp(Message from ${MESSAGE(from)})
same => n,NoOp(Message to ${MESSAGE(to)})
same => n,NoOp(Body: ${MESSAGE(body)})
same => n,Set(dest=${EXTEN})
same => n,Set(sender=${CUT(MESSAGE(from),<,2)})
same => n,Set(sender=${CUT(sender,@,1)})
same => n,Set(sender=${CUT(sender,:,2)})
same => n,Set(text=${MESSAGE(body)})
same => n,Set(MESSAGE(body)=${sender}: ${text})
same => n,GotoIf($["${EXTEN}" = "4000"]?send_javier)
same => n,GotoIf($["${EXTEN}" = "4001"]?send_belen)
same => n,Hangup()

same => n(send_javier),MessageSend(sip:javier,Centro de Mensajes)
same => n,NoOp(Message status: ${MESSAGE_SEND_STATUS})
same => n,Hangup()

same => n(send_belen),MessageSend(sip:belen,Centro de Mensajes)
same => n,NoOp(Message status: ${MESSAGE_SEND_STATUS})
same => n,Hangup()
```

### Interactive Menu

```ini
[demo]
exten => s,1,Answer()
same => n,Set(TIMEOUT(digit)=5)
same => n,Set(TIMEOUT(response)=10)
same => n(menu),Background(main-menu)
same => n,WaitExten()

exten => 1,1,Playback(option-1)
same => n,Hangup()

exten => 2,1,Playback(option-2)
same => n,Goto(s,menu)

exten => i,1,Playback(invalid)
same => n,Goto(s,menu)

exten => t,1,Playback(goodbye)
same => n,Hangup()
```

## Hints and Presence

Hints enable presence/busy lamp field (BLF):

```ini
[suscribir]
exten => 4000,hint,SIP/javier
exten => 4001,hint,SIP/belen
```

## Applying Changes

After modifying `extensions.conf`, reload the dialplan:

```bash
asterisk -rx "dialplan reload"
```

Or from the Asterisk CLI:

```
asterisk> dialplan reload
```
