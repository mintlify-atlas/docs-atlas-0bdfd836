## Welcome to Asterisk Tutorial

This comprehensive tutorial will guide you through the complete process of installing and configuring **Asterisk PBX** on Debian 9.11. Whether you're new to PBX systems or looking to expand your telephony infrastructure knowledge, this guide provides step-by-step instructions for setting up a fully functional call center environment.

<Note>
This tutorial was created as a learning resource and includes real-world implementations of common PBX features. While this is a beginner-friendly guide, some errors may exist as this represents hands-on learning with Asterisk.
</Note>

## Key Features Covered

This tutorial demonstrates how to implement the following PBX features:

- **Voicemail System** with Spanish language support
- **Call Transfer & Redirection** between extensions
- **SIP Messaging** center for text communication
- **Group Calls** with call distribution
- **Call Pickup** functionality for shared extensions
- **Text-to-Speech** using Festival TTS engine
- **TLS Call Encryption** for secure communications
- **MariaDB Integration** for scalable data storage
- **DUNDi Protocol** for PBX interconnection
- **Business Hours** management and routing
- **Call Logging** and monitoring

## Technical Environment

<Info>
**System Requirements:**
- Debian 9.11 (GNU bash 4.4.12)
- Asterisk 15.7.1
- VMware Workstation 14 Player (Bridged network mode)
- MariaDB for database integration
- Festival for text-to-speech
</Info>
