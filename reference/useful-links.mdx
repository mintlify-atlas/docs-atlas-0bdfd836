## Official Documentation

### Asterisk Resources

- **[Asterisk: The Definitive Guide](https://bit.ly/2Hi7HNW)** - Comprehensive book covering Asterisk from basics to advanced topics
- **[Asterisk Wiki](https://wiki.asterisk.org/wiki/display/AST/Home)** - Official Asterisk documentation and guides
- **[Asterisk Community](https://community.asterisk.org)** - Community forums for questions and discussion
- **[VOIP-Info](https://www.voip-info.org/)** - Extensive VoIP and Asterisk reference documentation

### Protocol Documentation

- **[DUNDi Protocol](http://www.dundi.info/)** - Distributed Universal Number Discovery protocol information
- **[SIP (RFC 3261)](https://tools.ietf.org/html/rfc3261)** - Session Initiation Protocol specification
- **[RTP (RFC 3550)](https://tools.ietf.org/html/rfc3550)** - Real-time Transport Protocol specification

### Development and Editing

### Network and File Transfer

- **[MobaXterm](https://mobaxterm.mobatek.net/)** - Enhanced terminal for Windows with SSH, SFTP, and more
- **[FileZilla](https://filezilla-project.org/)** - Free SFTP/FTP client for file transfers
- **[PuTTY](https://www.putty.org/)** - Lightweight SSH and telnet client for Windows

### Audio Processing

- **[SoX (Sound eXchange)](http://sox.sourceforge.net/)** - Command-line audio processing tool

### Testing and Debugging

- **[Wireshark](https://www.wireshark.org/)** - Network protocol analyzer for debugging SIP/RTP
- **[sngrep](https://github.com/irontec/sngrep)** - Terminal-based SIP packet viewer
- **[SIPp](http://sipp.sourceforge.net/)** - SIP protocol test tool and traffic generator
