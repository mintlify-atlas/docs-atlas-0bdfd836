## What is Asterisk PBX?

Asterisk is an open-source [Private Branch Exchange (PBX)](https://en.wikipedia.org/wiki/Business_telephone_system#Private_branch_exchange) framework that enables you to build communications applications. It provides a complete telephony system that can handle voice calls, video calls, messaging, and more.

This tutorial was developed as part of a university project and has been shared to help others learn how to configure Asterisk from scratch.

<Note>
This is a beginner-friendly guide. While there may be multiple ways to configure Asterisk, this tutorial provides a solid foundation for understanding the core concepts.
</Note>

## Purpose of This Tutorial

This tutorial guides you through:

- Installing and configuring Asterisk on Debian
- Setting up basic PBX functionality
- Implementing advanced features (voicemail, call distribution, queues)
- Integrating Asterisk with MariaDB
- Building a complete Call Center simulation

The tutorial is structured in small modules to make it easier to understand concepts and test each integration step by step.

## Tutorial Structure

The tutorial is organized into the following main sections:

## Nomenclature

Throughout this tutorial, commands are prefixed with specific symbols to indicate where they should be executed:

<CodeGroup>
```bash $ - Linux Terminal
$ [command]
```

```vim : - Vim Editor
: [command]
```

```asterisk CLI> - Asterisk Console
CLI> [command]
```

```sql MariaDB> - MariaDB Database
MariaDB> [command]
```
</CodeGroup>

<Info>
**Command Prefix Guide:**
- **$** (dollar sign) - Linux terminal commands
- **:** (colon) - Vim editor commands
- **CLI>** - Asterisk CLI (Command Line Interface)
- **MariaDB>** - MariaDB database console
- **Quoted text** - Plain text content for configuration files
</Info>
